# Project: Calculating Final Velocities and Displacements Using SUVAT

## Overview:
Very straightforward project using $$v = u + at$$ and $$s = ut + \frac{1}{2}at^2$$ 
to calculate $$v$$ and $$s$$ as a refresher to Python.

## Usage:
Run `solver.py` and input initial velocity ($$u$$), acceleration ($$a$$), 
and time ($$t$$). The Solver returns final velocity ($$v$$) and 
displacement ($$s$$).

**Example:**
u = 0 m/s, a = 9.8 m/s², t = 2s
→ v = 19.6 m/s, s = 19.6 m
