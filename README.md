# Python for Sciences 🧪 (Completed)

## Introduction
This project was inspired by my A-Level Physics and Chemistry syllabus, in which I wanted to display various concepts using Python, with the Pendulum Analysis directly linking to one of my Physics presentations.

## 📂 Repository Structure
```text
.
├── 01_SUVAT/
│   ├── README.md              # Basics
│   └── solver.py              # Starter project to calculate SUVAT
├── 02_Pendulum_Analysis/
│   ├── README.md              # Physics SHM & Logic Paradox analysis
│   └── logic.py               # Core pendulum motion engine
└── 03_Titration_Master/
    ├── README.md              # Chemical equilibrium & NumPy specs
    ├── engine.py              # pH calculation with NumPy and Matplotlib
    ├── titration_curve.png    # Example output of the pH curve
    └── titration_oop.py       # Refactor of the initial engine using OOP
``` 
## Outline of Each Folder
1. [SUVAT](./01_SUVAT) - First Python project to calculate the final velocity using basic kinematics.

2. [Pendulum Analysis](./02_Pendulum_Analysis) - SHM modelling, comparing ideal and actual time periods.

3. [Titration Master](./03_Titration_Master/) - Generates a pH titration curve with a weak/strong acid against a strong base, then converted the code to an OOP structure.

## Conclusion for this Project
Started with Python maths and NumPy; I then progressed to OOP structures, which gradually led me to more complex, realistic programming.
